本文面向硬禾科技 eZ-PLM 电子元器件全生命周期管理平台，设计并实现了一套融合大语言模型 LLM 与工程规则推理的智能元器件选型与风险评估 Agent 系统，实现了从自然语言需求输入到 BOM 选型清单、供应链风险评估报告和电路拓扑分析报告输出的端到端自动化流程。系统采用 Pipeline 与 ReAct Agent 双模架构，在保证批量分析效率的同时兼顾多轮交互推理能力。

在数据集成方面，系统完成了 eZ-PLM 实时 API 的全链路接入，构建了基于制造商型号前缀的分组查询机制，并结合 HMAC-SHA256 签名认证实现安全访问。通过多前缀检索、五维约束过滤以及基于MPN的固定输出电压推断方法，实现了对真实在产器件数据的准确召回与筛选。

在需求理解方面，针对工程文本表述多样、领域术语复杂等特点，提出四级容错需求解析链，将 LLM 语义解析与规则校验相结合，实现了类别识别、拓扑判断、电压电流参数提取及等级偏好识别等关键任务的高准确率解析。实验结果表明，系统在 28 条评测用例上实现了 100\% 的解析通过率，验证了该方法在工程场景中的鲁棒性与可靠性。

在智能评分方面，构建了面向参考设计知识增强的双模自适应评分框架。当参考设计数据可用时，系统引入应用场景适配度与设计成熟度两个 LLM 软评分维度；当外部知识缺失时，则自动回退至纯规则评分模式，从而兼顾评分结果的稳定性与知识增强能力。同时，通过核心型号去重、动态约束归一化及推荐分级策略，提高了推荐结果的可读性与工程实用性。

在风险评估方面，提出规则验证与 LLM 叙述相分离的双层风险评估架构。规则引擎负责供应链与工程风险的确定性识别，LLM 仅用于生成工程背景说明，从机制上避免了风险结论受到语言模型不确定性的影响。

在系统架构方面，建立了以 \texttt{PartIR}、\texttt{ScoreBreakdown}、\texttt{TopologyIR} 和 \texttt{RiskIR} 为核心的标准化中间表示体系，实现了模块间的数据解耦与统一交互。系统同步生成结构化 JSON 数据与标准化工程文档，可直接支持工程评审及 eZ-PLM 平台的数据互通需求。

此外，系统基于 ChromaDB 构建了面向电源设计场景的工程知识库，并结合 LangChain 实现 ReAct Agent 多轮交互模式，使知识检索、替代器件分析及报告生成等能力能够以自然语言方式被统一调用，进一步提升了系统的智能化水平与工程应用价值。

实验结果表明，所提出的方法能够有效提升电子元器件选型与风险评估过程的自动化程度、决策效率和结果可解释性，为 LLM 在电子设计自动化与产品生命周期管理场景中的应用提供了一种具有工程可行性的实现方案。
