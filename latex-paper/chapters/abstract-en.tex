Electronic component selection is a critical, labor-intensive, and high-risk stage in hardware development. Traditional manual workflows require engineers to examine multidimensional parameters across numerous part numbers. As a result, they are often time-consuming and insufficient for systematic supply chain risk assessment and rapid generation of standardized engineering documents. To address this issue, this paper develops an intelligent agent system for electronic component selection and risk assessment targeting the eZ-PLM platform. The system is built on collaborative reasoning between a large language model (LLM) and an engineering rule engine. It employs a five-stage pipeline consisting of natural language requirement parsing, real-time API retrieval, hybrid scoring, evidence chain construction, and standardized report generation. In addition, it integrates a LangChain-based ReAct interaction mode to support multi-turn dialogue and dynamic decision-making.

The main technical contributions of the system are reflected in five aspects. First, a grouped query strategy for manufacturer part numbers is proposed based on the prefix search capability of eZ-PLM, allowing one query to retrieve over 200 active components. Second, a four-level requirement parsing chain is designed to handle six engineering issues, including parameter unit misidentification, missed negation patterns, and LLM topology hallucination. Third, an LLM hybrid scoring mechanism is constructed using eZ-PLM reference design data as contextual constraints. It evaluates parameter matching, supply chain stability, application suitability, and design maturity, and switches to a rule-driven two-dimensional scoring method when reference data are insufficient. Fourth, a risk assessment framework is established to keep rule-verified facts separate from LLM-generated descriptions. It covers 13 categories of deterministic risk and ensures the rigor and traceability of risk reports. Fifth, standardized intermediate representations, including PartIR, ScoreBreakdown, TopologyIR, and RiskIR, are defined to generate bill of materials (BOM) selection lists, risk assessment reports, and circuit topology analysis reports.

Experiments show a 92.9\% pass rate on 28 test cases and 100\% accuracy in parsing core parameter fields. Combined with a RAG knowledge base and ReAct interaction mode, the system further enhances knowledge coverage and interaction flexibility.
