#### 1.3 白名单覆盖测试 ✅
- **状态**: 完成
- **关键修复**: UTF-8 字符编码问题
  - **问题**: 请求体包含中文字符时返回 400 错误
  - **根本原因**: Pydantic BaseModel 在处理特定 UTF-8 编码的 JSON 时存在问题
  - **解决方案**: 将 `/analyze` 和 `/replacement` 端点改用 `Dict[str, Any] = Body(...)` 方式，手动处理 JSON 解析
  - **验证**: 所有中文输入测试用例通过（车规、国产等）
  - **代码变更**: app/main.py (commit 866a609)
