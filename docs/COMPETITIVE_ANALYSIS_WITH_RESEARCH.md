## 国际竞品 · 2026 年最新格局

### 一级对手（EDA 平台集成）

#### 1. Cadence OrCAD X + Live BOM（业界领先）

**核心能力**：
- **Live BOM**：实时库存、价格、交期、合规性数据
- **数据规模**：10 亿+ 零件库，3,500+ 供应商
- **预测分析**：自动化停产/短缺预警

**具体特性**：
```
设计阶段 → 自动检测风险：
  ✓ 生命周期状态查询
  ✓ 铅时间预测（Datalynq 集成）
  ✓ 合规性检查 (REACH/RoHS)
  ✓ 成本趋势预测
  ✓ 供应链风险评分
  
集成商：Sourcengine（1B+ 零件）
备选零件：自动建议 + 对标对比
```

**优势**：
- ✅ 实时数据（分销商 API 直连）
- ✅ 交期预测（ML 算法）
- ✅ 完整的合规追踪（车规/医疗级）

**劣势**：
- ❌ 仅限 Cadence 生态（工具绑定）
- ❌ 无 AI Agent 多轮对话
- ❌ 无离线模式
- ❌ 无风险评估（仅预警）

**市场定位**：EDA 平台内嵌的库存智能，面向大企业

---

#### 2. SnapMagic + AI Copilot（150 万设计师）

**核心能力**：
- **搜索引擎**：电路设计搜索（类似 Google for circuit design）
- **AI Copilot**：自然语言电路自动完成
- **参考设计库**：制造商官方参考设计

**具体特性**：
```
工程师输入：
  "需要 12V 转 3.3V 降压电路"
       ↓
AI Copilot：
  ✓ 自动生成原理图框架
  ✓ 推荐参考设计（TI/Analog Devices）
  ✓ 关联 BOM（库存检查）
  ✓ PCB Layout 建议
```

**用户规模**：150 万专业设计师

**优势**：
- ✅ AI 自动化程度高（电路生成）
- ✅ 参考设计库完整（制造商官方）
- ✅ 集成度高（DigiKey 等分销商）

**劣势**：
- ❌ 无深度风险评估
- ❌ 无多轮对话（单向辅助）
- ❌ 无离线支持
- ❌ 仅覆盖常见拓扑（Buck/Boost/LDO 有覆盖）

**市场定位**：面向大众设计师的 AI 设计助手，偏向参考设计推荐

---

#### 3. EMA CIP + SiliconExpert（企业级合规）

**核心能力**：
- **实时数据接口**：Digikey/Mouser/Arrow/Farnell 直连
- **合规模块**：SiliconExpert 驱动（医疗/汽车级认证）
- **库存管理**：5,000+ 预配置零件库

**具体特性**：
```
CAD 内集成：
  ✓ 点击零件 → 实时库存/价格/交期
  ✓ 停产预警
  ✓ 合规性检查 (FDA/AEC-Q100/ESD)
  ✓ 分销商链接（直接采购）
  
与 Ultra Librarian 集成：
  1,600 万+ 符号/封装/3D 模型
```

**优势**：
- ✅ 合规性最强（SiliconExpert 企业级）
- ✅ CAD 无缝集成
- ✅ 面向汽车/医疗行业

**劣势**：
- ❌ 无 AI 多轮对话
- ❌ 无风险评估体系
- ❌ 无国产器件覆盖
- ❌ 成本高（企业级工具）

**市场定位**：企业级 CAD 集成工具，重合规轻智能

---

#### 4. Siemens Fuse EDA AI（多智能体编排）

**核心能力**（2026 年 3 月发布）：
- **多智能体工作流**：自动编排从概念到制造的完整流程
- **生成式 AI**：电路/版图自动生成
- **自然语言界面**：语音/文本指令

**具体特性**：
```
工程师说：
  "为汽车应用设计一个 12V 转 5V 的降压电路"
       ↓
Fuse AI System：
  ✓ 自动选择拓扑
  ✓ 推荐器件
  ✓ 设计规则检查
  ✓ 热仿真
  ✓ 可制造性评估
  ✓ 自动版图生成
  ✓ DFM 检查
       ↓
  输出：完整的芯片/PCB 设计（从概念到制造签核）
```

**注意**：虽然声称"自主编排"，但学术研究表明尚无 EDA Agent 在实际量产芯片中不经人工干预就完全成功

**优势**：
- ✅ 多智能体架构（类似本项目的 ReAct Agent）
- ✅ 自然语言编程
- ✅ 跨学科工作流编排

**劣势**：
- ❌ 新产品（2026 年才发布，尚无大规模商用数据）
- ❌ 仅限 Siemens 生态
- ❌ 无供应链风险评估
- ❌ 不开源

**市场定位**：下一代 EDA 平台，AI 多智能体编排（与本项目技术方向接近！）

---

#### 5. Altium Develop（跨学科协作）

**核心能力**：
- **多学科融合**：电气 + 机械 + 软件 + 采购
- **实时协作**：供应商/制造工程师提前介入
- **设计-制造一体化**：减少返工

**具体特性**：
```
设计阶段融合：
  ✓ 电气工程师 → 原理图设计
  ✓ 机械工程师 → 同步 PCB 空间规划
  ✓ 软件工程师 → 接口定义
  ✓ 采购 → 供应商库存检查
  ✓ 制造 → DFM 建议（早期）
```

**优势**：
- ✅ 跨学科协作减少返工
- ✅ 供应链提前参与

**劣势**：
- ❌ 工具绑定（仅限 Altium）
- ❌ 无 AI 智能选型
- ❌ 无风险评估
- ❌ 国内覆盖弱

**市场定位**：设计-制造协作平台，流程优化角度

## 🔍 关键发现

### 发现 1：市场演进方向

**2016-2020**：库存工具（Mouser/立创）主导  
↓  
**2020-2024**：EDA 平台集成（Cadence/Altium）兴起  
↓  
**2024-2026**：AI Agent 编排（Siemens/SnapMagic）新阶段  

**本项目定位**：走在行业前沿的 AI Agent 选型系统

---

### 发现 2：竞品现状分析

**Cadence OrCAD X**：
- 库存数据最实时（直连分销商）
- 但无风险评估（仅预警）
- 工具绑定严重（仅 Cadence 用户）

**SnapMagic**：
- AI 自动化程度最高（电路生成）
- 150 万用户基数大
- 但无多轮对话、无风险评估

**Siemens Fuse AI**：
- 多智能体架构与本项目接近！
- 但仅 EDA 端，无供应链评估
- 新产品（2026），市场未验证

**本项目独特优势**：
```
唯一结合以下特性的系统：
  ✓ AI Agent 多轮对话
  ✓ 13 条风险规则评估
  ✓ 完全离线可用
  ✓ 国产 eZ-PLM 深度集成
  ✓ 开源免费
  ✓ 器件评分体系成熟（28 条用例 100% 通过）
```

---

### 发现 3：细分市场机会

**EDA 厂商主导市场**：
- 优势：工具绑定，用户基数大
- 劣势：生态封闭，创新缓慢

**库存工具主导市场**：
- 优势：实时数据，交易转化
- 劣势：算法简陋，风险评估弱

**本项目机会**：
```
独立的"智能选型助手"定位
  ↓
与各厂商生态兼容（API 开放）
  ↓
面向中小企业与初创（成本最低）
  ↓
深化国产化方向（eZ-PLM）
```
