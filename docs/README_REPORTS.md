## 🔧 PDF 生成方式

两个 PDF 报告由 Python 脚本自动生成：

### 生成脚本
- `generate_pdf_report.py` - 生成标准版 PDF
- `generate_detailed_pdf.py` - 生成详细版 PDF

### 运行方式
```bash
# 生成标准版 PDF
python generate_pdf_report.py

# 生成详细版 PDF  
python generate_detailed_pdf.py

# 生成两个版本
python generate_pdf_report.py && python generate_detailed_pdf.py
```

### 依赖库
- reportlab - PDF 生成库
- 可通过 `pip install reportlab` 安装
